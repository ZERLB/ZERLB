## ⚡ About Me

I focus on **Full-Stack** development, creating robust APIs on the Backend and highly functional interfaces on the Frontend. I design efficient databases and build integrations that solve real-world problems, thoroughly enjoying the optimization of workflows and IT infrastructures.

- 🚀 Currently developing high-impact **web architectures and Full-Stack systems**. Leading the creation of the **Institutional University Management System (SIGU)** and my core project **NEXUS**, focused on innovative system and network integrations.
- 🐧 My main development and administration environment is **Ubuntu/Linux**, managing repositories, servers, and virtual machines using tools like **XCP-ng / XenCenter**.
- 🤖 I continuously boost my productivity and write code using artificial intelligence and cutting-edge tools such as **Cursor IDE**, **Warp Terminal**, **Claude**, and **Antigravity AI**. All carefully structured with databases in **DBeaver** and documentation in **Notion**.
- 💬 Ask me about: **Python, Express, PHP, Linux, virtualization, and AI workflow integrations**.
